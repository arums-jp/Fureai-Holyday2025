## 2. 課題・問題点の指摘（最も多い）

休暇取得困難や制度設計への疑問が多数寄せられた。

### 代表的な意見

| № | 意見 |
|---|------|
| 1 | 「ほとんどの保護者の方が、ふれあいホリデーで仕事は休めないと聞きます。自分自身も職場が推奨していないので、この為に休みは取れないです。」 |
| 2 | 「子供だけ休みにしても親は休めない。この取組をするなら親も休みにしてほしい。」 |
| 3 | 「不定休の仕事だと、ふれあいホリデー当日も、1人は仕事になるため、あまり意味のない休みだと思う。ふれあいホリデーと綺麗事を言いながらも、結局は教員の為の休みだと思う。」 |
| 4 | 「本当に迷惑なのでやめていただきたい。我が家の実態は子どもたちだけでダラダラとすごし、メディア三昧。親は昼ごはんの準備の負担増。」 |

### その他の声

- 「有給が少ない中、有給をつかって休暇を取得しないといけないのが苦しいです」
- 「子どもだけで家で過ごすことになり、お弁当の準備など逆に負担が増える」
- 「医療、介護従事者の親が、一斉に休んだらどうなるか想像して欲しい」
- 「行事が多い時期に実施されると会社を何度も休まないといけなくなる」

### 主要な課題

1. **休暇取得の困難**
   - 職場に制度が周知されていない
   - 有給休暇が足りない
   - 連休前は特に休みづらい

2. **子どもの過ごし方**
   - 子どもだけで留守番
   - 学童に預けざるを得ない
   - 昼食準備の負担増

3. **制度設計への疑問**
   - 誰のための休みか不明
   - 教員のための休みではないか
   - ふれあいの意味がない

## 3. 改善提案

建設的な改善案も多数寄せられた。

### 代表的な意見

| № | 意見 |
|---|------|
| 1 | 「ふれあいホリデーを市が実施するのであれば各企業などに呼びかけや休暇を取れるように働きかけをしてくれないと、ふれあいホリデーをやる意味がないです。」 |
| 2 | 「各家庭ごとに好きな日に1日お休みを取れるようにして欲しい。各企業もそれに合わせて、ふれあい休暇を作って、積極的に休暇を取れるようにして欲しい。」 |
| 3 | 「倉吉市だけやることでさまざまな歪みを生んでしまっています。鳥取県で、あるいはせめて中部地区で揃えてできないのであれば、やめるべきだと今年も感じました。」 |
| 4 | 「一斉に実施ではなく、地域別に時期をずらし、地域のイベントに参加がいいとおもう。」 |

### 提案の分類

#### 企業・職場への働きかけ
- 市から企業への周知・協力要請
- 休暇取得の義務化・推奨
- 企業への助成金制度
- 市内事業所への通達

#### 実施方法の変更
- 日にちを各家庭で選べるようにする
- 学校別・地区別で分散実施
- 1日ではなく期間内で選択制に

#### 実施範囲の拡大
- 鳥取県全体での実施
- 中部地区での統一実施
- 高校・保育園も同日に

#### 時期の変更
- 6月など祝日のない月へ
- テスト期間を避ける
- 連休にくっつけない

## 4. 廃止・中止要望

制度自体の廃止を求める声が非常に多い。

### 代表的な意見

| № | 意見 |
|---|------|
| 1 | 「これからもふれあいホリデー、実施するのですか？休みが取りづらい保護者からしたら、やめて欲しいです。」 |
| 2 | 「この休暇をやめて下さい。休めない親はただただ、心理的負担です。」 |
| 3 | 「必要ないのでやめてください」 |
| 4 | 「ふれあいホリデーは必要ない。普段から子供とはちゃんとふれあっているから大丈夫です。」 |

### 廃止要望の理由

- 休暇が取れないので子どもに申し訳ない気持ちになる
- 結局子どもだけで過ごすなら意味がない
- 昼食準備など親の負担が増えるだけ
- 普段から家族との時間は取れている
- 教員のための休みにしか見えない

## 総評

### 評価できる点
- 休暇を取得できた家庭からは概ね好評
- 平日で施設が空いている点は評価されている
- 子どもの喜ぶ姿を見られたという声も

### 深刻な課題

1. **休暇取得の構造的問題**
   - 企業・職場への周知が不十分
   - 休暇を取りたくても取れない現状
   - 同じ職場内で対象者が重なる問題

2. **不公平感の発生**
   - 休める家庭と休めない家庭の格差
   - 子どもへの申し訳なさ
   - ひとり親家庭への負担

3. **制度の本末転倒**
   - 「ふれあい」できないのに「ふれあいホリデー」
   - 昼食準備など親の負担増
   - 子どもだけで過ごす＝リスク増大

4. **アンケート反映への疑問**
   - 昨年も同様の意見が出ていたはず
   - 否定的意見も公表してほしいという声
   - 改善が見られないことへの不満

### 今後の検討課題

1. **企業への働きかけ強化**
   - 市から企業・事業所への正式な協力要請
   - 休暇取得促進のインセンティブ
   - 商工会等を通じた周知

2. **実施方法の抜本的見直し**
   - 分散実施（学校別・地区別）の検討
   - 各家庭で日程を選べる方式
   - または制度の廃止

3. **実施範囲の拡大検討**
   - 中部地区での統一実施
   - 県全体での実施

4. **休めない家庭への配慮**
   - 学童・イベントの充実
   - 昼食支援
   - 子どもの見守り体制
